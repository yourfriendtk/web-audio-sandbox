Web Audio Sandbox
=============

Setup:
  * Install nodemon:
  * `npm install -g nodemon`

To run this server:

  * `nodemon server.js`
  * Visit http://localhost:5000/

To view the server in action:

  * Visit http://blooming-wildwood-9666.herokuapp.com/ | git@heroku.com:blooming-wildwood-9666.git
